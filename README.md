# Collge Housing Rater (App name TBD)

## Description
Everyone has heard of college housing horror stories. Ranging from living with black mold, horrible RAs, disgusting bathrooms, miserable management and non-existent maitenance. Not to mention the countless other stories you've heard from previous or current students or have read online. Now, with all of social media and the internet, the days of living in the dark and being uninformed. This application was created to give insights, reviews, ratings, and information from students to students about collge's housing.

This application was created with students in mind. Going to college is already a daundting journey, so choosing where you are going to live on campus shouldn't add to that. Whether you are a studen going to college in another state, or a student staying in town. Picking where you are going to live should be an easy decision, made via informative and transparent housing reviews, history, and ratings from other fellow students, past and present. 

This application aims to do just that. Allow students to review and rate college affiliated housings and allow students to review these ratings and reviews in order to be informed and choose the right place to live. This application also aims to help keep universities accountable in order for them to keep their housing up-to-date and appropriate for all students to live comfortably in.

## Tech
- Backend: Java, Spring Boot, Apache Maven
- Frontend: React.js, HTML, CSS
- Database: PostgreSQL (Containerized using Docker)
- Version Control: Git/GitHub
- Cloud Hosting: AWS
- Auth: Spring Security
